# agentic-ai-workflow
This project demonstrates an agent-based AI system where multiple agents collaborate to solve complex tasks through structured reasoning and execution.

The system takes a user input task and processes it through two main agents:

Planner Agent → Breaks down the task into smaller, manageable steps

Executor Agent → Executes each step sequentially and generates the final output

This architecture mimics real-world agentic AI systems used in automation, decision-making, and large language model orchestration.
